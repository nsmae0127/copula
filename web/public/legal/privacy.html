2. 개인정보 이용 목적

  • 초대 기반 커뮤니티 생성, 참여, 멤버 권한 관리
  • 일정, 공지, 앨범, D-Day, 관계 약속 기능 제공
  • 비밀번호 재설정, 계정 관리, 고객 문의 대응
  • 서비스 보안, 오류 분석, 부정 이용 방지
  • 이용자가 동의한 경우 푸시 알림 발송

3. 보관 기간

회원 탈퇴 또는 커뮤니티 삭제 시 관련 개인정보는 지체 없이 삭제합니다. 다만 법령상 보관 의무가 있거나 분쟁 대응을 위해 필요한 정보는 해당 기간 동안 분리 보관할 수 있습니다.

4. 제3자 제공 및 처리 위탁

Copula는 이용자 동의 또는 법령상 근거가 있는 경우를 제외하고 개인정보를 제3자에게 제공하지 않습니다. 서비스 운영을 위해 다음 처리자를 이용할 수 있습니다.

수탁자업무
Supabase인증, 데이터베이스, 스토리지, 서버리스 함수 인프라 제공
AppleApple 로그인 사용 시 인증 처리
브라우저/운영체제 푸시 서비스이용자가 허용한 푸시 알림 전달

5. 이용자의 권리

이용자는 개인정보 열람, 정정, 삭제, 처리정지, 동의 철회를 요청할 수 있습니다. 계정 정보는 앱의 계정 화면에서 직접 수정할 수 있으며, 탈퇴 또는 삭제 요청은 문의 채널로 접수할 수 있습니다.

6. 안전성 확보 조치

  • 인증 기반 접근 제어와 커뮤니티 멤버십 기준 Row Level Security 적용
  • 비공개 스토리지와 제한된 초대 코드 기반 참여
  • 비밀번호 및 인증 정보의 직접 저장 금지
  • 운영 접근 권한 최소화와 로그 기반 장애 대응
